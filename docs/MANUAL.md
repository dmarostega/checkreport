## 📖 Visão Geral

O **CheckReport5** é um sistema para criação de checklists, vistorias simples, relatórios operacionais e registros técnicos não regulados.

**Uso ideal para**:
- Técnicos de manutenção e instalação
- Vistoriadores informais
- Prestadores de serviço residencial
- Pequenas empresas que precisam documentar serviços
- Inspeções e auditorias internas

**Disclaimer Importante**: O CheckReport5 é uma ferramenta de organização e registro. Ele não substitui responsabilidade técnica, assinatura profissional, ART, laudo jurídico, laudo médico ou documento regulado. **A responsabilidade pelo conteúdo gerado é do usuário/profissional.**

## 🚀 Começando

### 1. Criar uma Conta

1. Acesse [CheckReport](https://checkreport.app)
2. Clique em **"Comece Gratuitamente"**
3. Preencha seus dados:
   - Email
   - Senha (mínimo 8 caracteres)
   - Confirme a senha
4. Clique em **"Criar Conta"**
5. Verifique seu email para confirmar a conta

### 2. Configurar seu Perfil

1. Após criar a conta, vá para **"Meu Perfil"**
2. Atualize suas informações:
   - Nome completo
   - Empresa/Negócio (opcional)
   - Telefone (opcional)
3. Clique em **"Salvar Alterações"**

## 👥 Gerenciando Clientes

Os clientes são as empresas ou pessoas para as quais você está fazendo o trabalho.

### Adicionar um Cliente

1. No dashboard, clique em **"Clientes"**
2. Clique em **"+ Novo Cliente"**
3. Preencha os dados:
   - **Nome**: Nome da empresa ou pessoa
   - **Email**: Email do cliente (opcional)
   - **Telefone**: Telefone para contato (opcional)
   - **Endereço**: Endereço completo
   - **Cidade**: Cidade
   - **Estado**: Estado/UF
   - **Notas**: Observações adicionais (opcional)
4. Clique em **"Salvar"**

### Editar ou Deletar um Cliente

1. Vá para **"Clientes"**
2. Clique no cliente desejado
3. Para editar: altere os dados e clique em **"Salvar"**
4. Para deletar: clique em **"Deletar"** e confirme

## 📋 Criando Modelos de Checklist

Os **modelos de checklist** são templates reutilizáveis que você pode usar várias vezes.

### Criar um Modelo

1. No dashboard, clique em **"Modelos de Checklist"**
2. Clique em **"+ Novo Modelo"**
3. Preencha os dados principais:
   - **Nome**: Ex: "Vistoria de Ar Condicionado"
   - **Descrição**: Ex: "Checklist completo para inspeção de sistema de AC"
   - **Status**: Ativo ou Inativo

### Adicionar Seções ao Modelo

As seções organizam o checklist em grupos lógicos.

1. No modelo criado, clique em **"+ Adicionar Seção"**
2. Digite o nome da seção: Ex: "Equipamentos"
3. Clique em **"Salvar"**

**Exemplos de seções para diferentes tipos de serviço**:

**Vistoria Elétrica**:
- Dados iniciais
- Inspeção visual
- Medições
- Equipamentos
- Resultados
- Recomendações

**Limpeza de Piscina**:
- Dados iniciais
- Inspeção visual
- Testes de água
- Limpeza realizada
- Fotos antes/depois
- Observações

**Manutenção de Câmeras**:
- Dados do local
- Equipamentos presentes
- Estado dos cabos
- Funções testadas
- Problemas encontrados
- Ações recomendadas

### Adicionar Campos ao Modelo

1. Dentro de uma seção, clique em **"+ Adicionar Campo"**
2. Configure o campo:
   - **Tipo**: escolha o tipo de resposta
   - **Pergunta**: escreva a pergunta ou label
   - **Descrição**: (opcional) dica para quem está preenchendo
   - **Obrigatório**: marque se é necessário responder
3. Clique em **"Salvar"**

#### Tipos de Campos

| Tipo | Uso | Exemplo |
|------|-----|---------|
| **Texto** | Respostas abertas | "Qual é o modelo do equipamento?" |
| **Número** | Valores numéricos | "Qual a temperatura medida? (°C)" |
| **Sim/Não** | Decisões binárias | "O equipamento está funcionando?" |
| **Seleção** | Múltiplas opções | "Estado: Bom / Regular / Ruim" |
| **Data** | Datas específicas | "Data da última manutenção" |
| **Foto** | Evidência visual | "Foto do problema encontrado" |
| **Observação** | Texto longo | "Descreva em detalhes o problema" |

## 📝 Preenchendo um Checklist

Uma vez que você criou um modelo, pode usá-lo para preencher relatórios.

### Criar um Novo Relatório

1. No dashboard, clique em **"Relatórios"**
2. Clique em **"+ Novo Relatório"**
3. Escolha um **modelo** (será exibida a lista de seus modelos ativos)
4. Escolha um **cliente**
5. Clique em **"Criar"**

### Preenchendo o Relatório

1. O sistema exibirá as seções e campos do modelo
2. Para cada campo:
   - **Texto**: Digite a resposta
   - **Número**: Insira o valor numérico
   - **Sim/Não**: Clique em "Sim" ou "Não"
   - **Seleção**: Escolha uma das opções
   - **Data**: Clique para abrir o calendário
   - **Foto**: Clique para fazer upload de uma imagem
   - **Observação**: Digite texto longo sem limite

3. Você pode **salvar como rascunho** a qualquer momento
4. Após preencher tudo, clique em **"Concluir"**

### Adicionando Fotos

Durante o preenchimento:

1. Clique no campo **"Foto"** ou na seção de fotos
2. Escolha a imagem no seu computador ou celular
3. A foto será enviada e exibida no relatório
4. Você pode adicionar **múltiplas fotos** por relatório

**Dica**: Use fotos para documentar:
- Estado inicial do equipamento
- Problemas encontrados
- Antes e depois de correções
- Etiquetas ou números de série

## 📊 Status dos Relatórios

Cada relatório possui um status que indica seu andamento:

| Status | Significado |
|--------|-------------|
| **Rascunho** | Ainda está sendo preenchido |
| **Concluído** | Preenchimento finalizado, pronto para compartilhar |
| **Enviado** | Foi enviado para o cliente |
| **Arquivado** | Arquivado para referência futura |

## 📄 Gerando e Compartilhando Relatórios

### Gerar PDF

1. Abra um relatório **"Concluído"**
2. Clique em **"Exportar PDF"**
3. O arquivo será baixado automaticamente
4. O PDF incluirá:
   - Todas as respostas
   - Fotos anexadas
   - Data e cliente
   - Seu nome e empresa

### Compartilhar Link Público

1. Abra um relatório **"Concluído"**
2. Clique em **"Gerar Link Público"**
3. Um link será gerado automaticamente
4. Copie e compartilhe com o cliente
5. O cliente pode visualizar o relatório sem fazer login
6. O link é protegido por token e não pode ser indexado por buscadores

**Dica**: Use links públicos para:
- Enviar relatório por email
- Integrar em seu website
- Compartilhar com múltiplos clientes
- Armazenar como referência

## ❓ Perguntas Frequentes

### Posso usar CheckReport em offline?

Não, CheckReport requer conexão à internet. Recomenda-se ter um backup de seus dados.

### Quantos relatórios posso criar?

Depende do seu plano:
- **Gratuito**: 10/mês
- **Pro**: 200/mês
- **Plus**: Ilimitado

### Posso deletar um relatório?

Sim, relatórios podem ser deletados a qualquer momento. Use com cuidado!

### Como faço para exportar meus dados?

Entre em contato conosco em support@checkreport.app para solicitar exportação de dados.

### O relatório tem validade jurídica?

**Não**. CheckReport5 é uma ferramenta de registro. Não substitui documentos regulados, assinaturas eletrônicas avançadas ou laudos de engenharia.

### Posso integrar com meu sistema?

Futuras versões terão API REST para integrações. Acompanhe nosso roadmap.

### Quanto tempo os dados ficam armazenados?

Dados são armazenados enquanto sua conta estiver ativa. Contas inativas por 1 ano podem ser deletadas.
