# CheckReport5

SaaS para criação de checklists, vistorias simples, relatórios operacionais e registros técnicos não regulados para prestadores de serviço.

## Stack

- Laravel 13
- Vue.js 3
- TypeScript
- MySQL
- Tailwind CSS
- Vite
- Inertia.js

## Objetivo

Permitir que prestadores criem modelos de checklist, preencham relatórios, adicionem fotos, registrem informações de execução e gerem relatórios simples.

## Funcionalidades previstas

- Autenticação
- Dashboard
- CRUD de clientes
- CRUD de modelos de checklist
- Seções e campos personalizados
- Preenchimento de checklists
- Upload de fotos
- Geração de relatório
- Link público de visualização
- Painel admin
- CRUD de planos
- Configurações globais
- SEO
- robots.txt
- sitemap.xml
